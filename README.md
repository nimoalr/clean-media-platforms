# Clean Media Platforms

A browser extension that removes annoying upsells and forces sane defaults on social media.

Works on Chrome, Edge, Brave, Arc, Opera, and any Chromium-based browser.

## What it does

**YouTube**
- Hides Shorts from the homepage
- Hides the "More from YouTube" sidebar (Premium, Music, Kids)

**X (Twitter)**
- Hides the "Upgrade to Premium+" upsell
- Defaults to the Following tab with Recent sort
- Swaps the tab order so Following comes first

**Instagram**
- Defaults to the Following feed on the homepage
- Swaps the feed tab order so Following comes first
- Fixes the sidebar avatar being clipped behind the header

## Install

1. Download the latest `clean-media-platforms.zip` from [Releases](../../releases/latest)
2. Unzip it somewhere permanent
3. Go to `chrome://extensions` (or `edge://extensions`, `brave://extensions`, etc.)
4. Enable **Developer mode**
5. Click **Load unpacked** and select the unzipped folder
6. Done — refresh your open tabs
